# Front-end
Overview

HomeHub is a smart home automation application that allows users to remotely control household electronics such as lights, fans, televisions, air conditioners, and power outlets from a mobile device.

This repository contains the front-end (mobile application) that provides the user interface for interacting with the HomeHub system. The app communicates with the backend server and IoT devices to send commands and display real-time device status and energy usage.

Features

Remote control of home appliances

Real-time device status updates

Electricity usage monitoring

User authentication (login/signup)

Device management (add, remove, rename devices)

Secure communication with backend services

User-friendly interface
